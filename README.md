# 🌱 식물 광합성 시뮬레이터

> 빛, 이산화탄소, 물, 온도, 빛 파장 등 조건을 조절하며 광합성 과정을 인터랙티브하게 학습하는 교육용 시뮬레이터

https://photosynthesis-simulator-two.vercel.app/

## 🎯 주요 기능

- **조절 가능한 환경 조건**: 빛 세기, 이산화탄소 농도, 물 양, 온도, 빛 파장(적색/녹색/청색)
- **과학적 시뮬레이션 엔진**: Michaelis-Menten 포화 곡선 + Liebig's Law 제한 인자 모델
- **온도 효과**: 효소 활성 종 모양 곡선 반영 (최적 온도 25~35°C)
- **빛 파장 흡수**: 엽록소 흡수 스펙트럼 (적색광 660nm / 녹색광 530nm / 청색광 430nm)
- **5종 프리셋 시나리오**: ☀️ 맑은 날 / ☁️ 흐린 날 / 🏡 온실 / 🏜️ 사막 / 🌊 심해
- **고품질 Canvas 시각화**: 파티클 글로우, 엽록체 그라나 구조, O₂ 버블, ATP 다이아몬드, 포도당 육각형
- **실시간 제한 인자 표시**: 현재 가장 제한적인 요소를 실시간으로 안내
- **반응 단계 시각화**: 광반응(빛 의존) → 탄소동화(Calvin Cycle)
- **실시간 그래프**: 산소(O₂) 및 포도당(C₆H₁₂O₆) 생성량 추적
- **시뮬레이션 조절**: 재생/일시정지, 반응 속도 조절
- **반응형 UI**: 데스크톱(3컬럼) / 태블릿(2컬럼) / 모바일(1컬럼) 대응

## 🧪 반응 수식

```
6CO₂ + 6H₂O + 빛 에너지 → C₆H₁₂O₆ + 6O₂
```

## 🔬 시뮬레이션 모델

### Michaelis-Menten 포화 곡선
각 환경 인자(빛, CO₂, 물)의 효율은 단순 비례가 아닌 포화 곡선으로 계산됩니다:

```
rate = Vmax × [S] / (Km + [S])
```

- `Km_light = 30`, `Km_co2 = 25`, `Km_water = 20`

### Liebig's Law (최소율의 법칙)
광합성 전체 속도는 가장 부족한 인자에 의해 결정됩니다:

```
전체 속도 = min(rate_light, rate_co2) × rate_water × tempFactor × wavelengthFactor
```

### 온도 효과 (효소 활성)
```
tempFactor = exp(-(T - 30)² / (2 × 10²))
```
최적 온도 30°C, 너무 낮거나 높으면 효율 급감

### 빛 파장 흡수율
| 파장 | nm | 엽록소 흡수율 |
|------|-----|--------------|
| 🔴 적색광 | 660 | 90% |
| 🔵 청색광 | 430 | 85% |
| 🟢 녹색광 | 530 | 10% |

## 📚 학습 포인트

### 광반응 (빛 반응)
- 광계II에서 빛을 흡수하여 물(H₂O)을 분해
- 산소(O₂) 방출, 전자 전달, ATP 합성

### 탄소동화 반응 (Calvin Cycle)
- 이산화탄소(CO₂) 고정
- 3-PGA → G3P → 포도당 합성

### 환경 요인과 광합성
- 빛 세기가 낮으면 광반응이 제한됨
- CO₂ 농도가 낮으면 탄소동화가 제한됨
- 물 부족은 전체 반응을 제한함
- 온도가 너무 낮거나 높으면 효소 활성 저하
- 녹색광은 엽록소에 거의 흡수되지 않음

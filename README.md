# 🌱 식물 광합성 시뮬레이터

> 빛, 이산화탄소, 물 등 조건을 조절하며 광합성 과정을 인터랙티브하게 학습하는 교육용 시뮬레이터

![Preview](https://raw.githubusercontent.com/sigco3111/blog-images/main/photosynthesis-simulator-preview.png)

## 🎯 주요 기능

- **조절 가능한 조건**: 빛 세기, 이산화탄소 농도, 물 양
- **반응 단계 시각화**: 광반응(빛 의존) → 탄소동화(Calvin Cycle)
- **실시간 그래프**: 산소(O₂) 및 포도당(C₆H₁₂O₆) 생성량 추적
- **시뮬레이션 조절**: 재생/일시정지, 반응 속도 조절

## 🧪 반응 수식

```
6CO₂ + 6H₂O + 빛 에너지 → C₆H₁₂O₆ + 6O₂
```

## 📚 학습 포인트

### 광반응 (빛 반응)
- 광계II에서 빛을 흡수하여 물(H₂O)을 분해
- 산소(O₂) 방출, 전자 전달, ATP 합성

### 탄소동화 반응 (Calvin Cycle)
- 이산화탄소(CO₂) 고정
- 3-PGA → G3P → 포도당 합성
